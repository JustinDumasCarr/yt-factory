# Decisions

This file records **why** we made important choices, to prevent rework and repeated debates.

Guidelines:
- Prefer fewer, higher-signal entries.
- Capture the context and tradeoffs, not just the conclusion.
- If a decision is reversed later, add a new entry linking to the old one.

---

## Template

### D### - Decision title
- Date: YYYY-MM-DD
- Status: proposed | accepted | superseded
- Context:
  - What problem were we solving?
- Decision:
  - What did we choose?
- Consequences:
  - What does this enable?
  - What constraints does this create?
- Alternatives considered:
  - Option A (why not)
  - Option B (why not)
